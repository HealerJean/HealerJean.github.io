# 1、Windows关闭进程

## 1.1、查看指定端口的占用情况

```
netstat -aon|findstr "9050"
```

## 1.2、查看PID对应的进程

```
tasklist|findstr "2016"
```

## 1.3、结束该进程

```
taskkill /f /t /im tor.exe
```
